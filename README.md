# Patrimeo CLI Tools

CLI tool for scaffolding Patrimeo addons. Built with Laravel Zero.

## Installation

```bash
composer install
```

## Usage

```bash
php patrimeo create-addon
```

Follows interactive prompts for slug, vendor, namespace, type, and destination.

## Addon Types

- `asset_balance` - Fetch and update asset balances
- `asset_transactions` - Fetch asset transactions
- `valuation` - Asset valuation

## Requirements

- PHP 8.2+
